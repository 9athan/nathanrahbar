Movement Activated Controller

Presence-aware media control for play/pause using simple sensors and microcontroller logic.

Embedded • UX • Hardware + Software

by Nathan Rahbar and Justin Herrera

Overview

This project combines hardware and software to interface with streaming apps (Spotify, Apple Music, YouTube, YouTube Music), enabling presence-based play/pause. The goal is to prototype, test, and refine a simple controller while building experience across embedded design and app control.

Project Specifications

  • User identification: Defines a specific user to avoid interruptions by others.
  • Use limitations: Not intended for collaborative spaces (e.g., study rooms).
  • Functionality: Focused on reliable play/pause only.
  • Environment: Best indoors and dry locations; not for outdoor/wet areas.

Future Improvements

  • UI polish: Clearer status/feedback and responsive layout.
  • Voice control: Voice commands with potential multilingual support.
